代码实现

简单代码实现

public static int[] zFunction(String s) {
    int n = s.length();
    int[] z = new int[n];
    for (int i = 1, l = 0, r = 0; i < n; i++) {
        if (i <= r) {
            if (z[i - l] < r - i + 1) {
                z[i] = z[i - l];
            } else {
                // z[i - 1] >= r - i + 1
                z[i] = r - i + 1;
                while (i + z[i] < n && s.charAt(z[i]) == s.charAt(i + z[i])) {
                    z[i]++;
                }
            }
        } else {
            // i > r
            while (i + z[i] < n && s.charAt(z[i]) == s.charAt(i + z[i])) {
                z[i]++;
            }
        }
        l = i;
        r = i + z[i] - 1;
    }
    return z;
}

精简代码实现

如上代码中内层 while 重复了两次，我们发现两个条件分支 i <= r && z[i - l] >= r - i + 1 与 i > r 可以合并，这样 while 循环就只需要写一次了。

public static int[] zFunction(String s) {
    int n = s.length();
    int[] z = new int[n];
    for (int i = 1, l = 0, r = 0; i < n; i++) {
        if (i <= r && z[i - l] < r - i + 1) {
            z[i] = z[i - l];
        } else {
            // i <= r && z[i - l] >= r - i + 1 || i > r
            z[i] = Math.max(0, r - i + 1);
            while (i + z[i] < n && s.charAt(z[i]) == s.charAt(i + z[i])) {
                z[i]++;
            }
        }
        l = i;
        r = i + z[i] - 1;
    }
    return z;
}

观察简单代码实现我们发现仅当 i <= r && z[i - l] >= r - i + 1 或 i > r 时会触发内层 while 循环，并且当 i > r 时 while 循环就是在推动 向后移动；当 i <= r && z[i - l] >= r - i + 1 时，由于此时 即 ，因此随着 增长也是在推动 向后移动。

对于外层循环，只有一遍线性遍历。
